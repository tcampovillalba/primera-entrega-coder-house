# Portfolio — Tatiana Campo Villalba

Sitio web personal (portfolio) de **Tatiana Campo Villalba**, diseñadora multimedia
especializada en diseño de experiencias digitales. El objetivo del sitio es presentar
su perfil profesional, habilidades y proyectos, y ofrecer una vía de contacto.

## Descripción

Sitio estático construido con **HTML5 semántico** y **CSS3**, con enfoque en
accesibilidad (uso de `alt`, etiquetas `<label>`, jerarquía de encabezados) y en una
estructura clara mediante etiquetas semánticas (`<header>`, `<nav>`, `<main>`,
`<section>`, `<article>`, `<figure>`, `<footer>`).

## Estructura del proyecto

```
1er entrega/
├── index.html                  # Página de inicio (punto de entrada)
├── README.md                   # Este archivo
├── tatiana-campo-villalba.jpg  # Imagen del portfolio
├── assest/
│   └── style.css               # Hoja de estilos
└── pages/
    ├── sobre-mi.html           # Biografía y trayectoria
    ├── habilidades.html        # Servicios y habilidades
    ├── proyectos.html          # Proyectos destacados
    └── contacto.html           # Datos de contacto y formulario
```

## Páginas

- **Inicio** — presentación y qué hago.
- **Sobre mí** — biografía, trayectoria e intereses.
- **Habilidades** — servicios de diseño y datos.
- **Proyectos** — proyectos destacados con las herramientas utilizadas.
- **Contacto** — correo, ubicación y formulario de contacto.

## Cómo verlo

Abre `index.html` en cualquier navegador web y navega con el menú superior.
